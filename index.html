Hexo error and solution

Error infomation:

Solution:

find root label in …/Hexo/_config
add “/“
that’s all!

Tips:

  1. See Error infomation line 3
    check load_config.js line 34
    you find
    config.root = config.root.replace(/\/*$/, '/');
  2. usually if you don’t change the configuration file, there will be no errors.
    when you see errors, first think about what changes you have made.

PPT notes

Veröffentlicht am 2017-08-23

许岑老师的PPT教程

1. 不需要使用的功能：

  • SmartArt
  • 超链接
  • 艺术字
  • 自带设计和模板
  • 动画和切换中的华丽型

2. 幻灯片的分类：

  • 给人看的幻灯片（信息量大）
  • 给人讲的幻灯片（信息量小）

    如果你看见一张幻灯片，你不知道他要说什么，那么这才是给人讲的幻灯片。
    但是每张幻灯片经过你的解释后又能够完美的表现你的主题内容。

3. 配合幻灯片演讲的流程：

  • 写出文字稿，记熟
  • 确定背景是深色还是浅色
  • 素材准备：文字，图片，数据，视频
  • 制作幻灯片
  • 演练
  • 修改

4. 以正确的方式制作幻灯片：

  • 默认
  • 动画效果慎用
  • 幻灯片要有焦点，按次序展现你的内容

    写幻灯片的目的是为了进行更有效的沟通。幻灯片归根结底只是辅助工具。

5. 演讲人是关键：

  • visualise的内容更具有说服力和感染力
  • 演讲是一种戏剧性的表达，有表演的成分在内。参考穹顶之下柴静最后对着旋转的地球发出感慨。
  • 演讲前要多排练

6. 为什么要做幻灯片：

  • 说明性，说明某些内容
  • 艺术性， 旋转星球营造了舞台效果
  • 数据可视化
  • 文字转化成图片后更具可读性

7. 具体的制作技巧：

  • 简洁
  • 把幻灯片做成连环画
  • 一致性与完整性，幻灯片前后的风格要一致
  • 次序，只有一个焦点。参考柴静发达国家减煤曲线和中国用煤曲线
  • 留白，内容出现转折时留下一个空白页。留白也是一种过渡
  • 结构，结合演讲的内容。一分钟至少要两张幻灯片
  • 感情，幻灯片要流露出感情
  • 强弱，要在许多的幻灯片中表现出某一个方面的强
  • 和声，清唱的感染力远不如编曲的感染力
  • 套路和技巧
  • 构图，对称性构图，版式设计
  • 好的配色方案，参考大红灯笼高高挂
  • 注意使用参考线和对齐
  • 一页幻灯片中的元素分层形成立体感
  • 善用格式刷
  • 更美观的文字可以使用Photoshop
  • 如果担心设备上没有字体，那就吧把所有的文字转换成图片格式
  • 幻灯片内的所有内容都要和演讲的内容相关
  • 背景加了要和没加一样，采用浅色的背景。深色背景要配白字
  • 取色形成搭配的色彩
  • 图表的展示也要遵循次序的原则
  • 动画效果移动，缩放，擦出擦除
  • 动画要有顺序和组合
  • 同种内容可以使用叠化即淡入淡出
